# Bleach for Hugo

![screenshot](/images/screenshot.png?raw=true "Home")

Bleach is an HTML5 and Bootstrap based blog template for Hugo. It provides a simple and clean look for a modern and professional blog.

## Features

- Fully Responsive
- Bootstrap
- Lightweight
